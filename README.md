# Simple Social Media Platform

## About The Project

It's a simple RESTful API made for a hackathon. It uses Swagger UI.

- [MongoDB](https://www.mongodb.com/)
- [Express.js](https://expressjs.com/)

- [Swagger UI](https://swagger.io/tools/swagger-ui/)
